Internet Security 101

  • Remove Flash (unless you *really* need it)
  • Remove Java (unless you *really* need it)
  • Set plugins as 'click-to-play' in your browsers
  • Use different password for each website
  • Use a password manager (so you can realistically do ^^). Or even write them down and store safely.
  • Use two-factor authentication (2FA) wherever available
  • Use the following browser extensions:
    • uBlock Origin
    • HTTPS Everywhere
    • Privacy Badger
  • Use 'private mode' in your browser when appropriate
  • Keep your stuff up to date - programs and operating system
  • Use common sense (if available)
